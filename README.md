# SmallAngularApp

## Install dependencies

```console
make install
```

## Start application

The following command will launch a dev server in order to run the application. You can access to it through this url: `0.0.0.0:4200`.

```console
make start
```

## Useful commands

To have a list of all available commands:

```console
make
```
